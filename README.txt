ТЕМА: «Разработка программного обеспечения выявления в социальных сетях информативных признаков подготовки к проведению «цветных» революций».

Разработанное программное обеспечение предназначено для поиска, анализа и систематизации различного рода информации 
имеющие какие-либо предпосылки и информативные признаки, в рамках подготовки к проведению революционных действий, 
у определенных пользователей сети, использующие социальную сеть Twitter в качестве инструмента
для формирования определенных взглядов и управления массовым сознанием населения.

Порядок установки и дальнейшей эксплуатации программы:

1) Запустите Setup_Twitter_FSI.exe
2) Выполните стандартный процесс установки программы. (При необходимости можно выбрать путь для установки и создать ярлык на рабочем столе)
3) Выбранная директория пользователем в процессе установки добавится в переменную окружения PATH (необходимо для корректной работы программы)
4) Для использования полноценной версии приложения необходимо: включите VPN перед ее запуском.
5) Программа требует наличие установленного браузера Chrome.
6) После запуска приложения, откройте чат с ботом @Getinf_bot в "Telegram" и отправьте /start
7) Данные о ходе работы программы записываются в .log файл, в директории где установленно приложение.
8) Для удаления программы, воспользуйтесь unins.exe в директории где установленно приложение.


Нет необходимости скачивать и устанавливать дополнительные драйвера для браузера Chrome и модули для организации взаимодействия с Telegramm и Twitter. 
В случае отсутствия у пользователя VPN или аккаута в Telegramm, скачайте и установите DEMO_Setup_Twitter_FSI.exe. Программа работает через консоль,
порядок установки и эксплуатации программы аналогичен, за исключение отсутствия пунктов №4 и №6.
